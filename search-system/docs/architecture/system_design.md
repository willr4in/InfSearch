# Системный дизайн — кратко

Это документ описывает высокоуровневую архитектуру каркаса поисковой системы.

Компоненты:
- C++ ядро: индекс, поиск, токенизация
- Python web: интерфейс и утилиты сбора данных
- PostgreSQL: хранение метаданных и корпуса

Для оценки на «3» реализуется минимальный функционал: токенизация, построение булевого индекса, командная утилита и простая веб-обёртка для демонстрации.
