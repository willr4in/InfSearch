# Search System — каркас проекта

Минимальная docker-композиция для демонстрации поисковой системы (C++ app + Python web + PostgreSQL + Redis + Nginx).

Запуск:

1. Собрать и запустить все сервисы в фоне:

   make build
   make up

2. Остановить:

   make down

Примеры:
- Веб-приложение будет доступно на порту 80 (nginx проксирует на Flask на 8000).
- База данных инициализируется скриптами в `database/`.

Примечания:
- Для простоты все компоненты имеют минимальные заглушки. Дальнейшая реализация функций индексации и поиска выполняется в `src/`.
